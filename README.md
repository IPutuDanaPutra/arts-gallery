# Classical Art Gallery

A modern, immersive digital art gallery showcasing a curated collection of classical masterpieces. Built with React and Framer Motion, it features a fluid parallax scrolling experience and a premium glassmorphism navigation.

## Features

- **Parallax Scrolling**: A multi-column staggered layout that moves at different speeds for a depth-filled scrolling experience.
- **Glassmorphism Navigation**: A floating "island" filter bar with a liquid glass effect, allowing users to filter artworks by century.
- **Responsive Design**: Fully responsive layout that adapts gracefully from desktop d to mobile (down to 320px).
- **Curated Collection**: 100+ verified classical artworks with high-quality images and metadata.
- **Dark Mode Aesthetic**: A sleek, dark-themed interface that puts the focus on the art.

## Tech Stack

- **Framework**: [React](https://react.dev/) + [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Routing**: [React Router](https://reactrouter.com/)

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/IPutuDanaPutra/arts-gallery.git
    cd arts-gallery
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

3.  Run the development server:
    ```bash
    npm run dev
    ```

4.  Open your browser at `http://localhost:5173`.
